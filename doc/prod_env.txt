
Context file pointing to the lunch4you war:
	/etc/tomcat6/Catalina/localhost/lunch4you.xml


server: vs833.web4u.cz

Installation:

1) Stop tomcat
	service tomcat6 stop

2) Upload lunch4you.war to /home/lunch4you/tomcat6
	- upload must be done under "lunch4you" linux account, so that it can be accessed and unzipped by Tomcat

3) Delete tomcat work directory
	rm -r /var/lib/tomcat6/webapps/lunch4you

4) Start tomcat
	service tomcat6 start
